WeatherApp

This project was created for the sole purpose to play with fancy libraries and APIs. You can find it on GitHub.

Used libraries:

  • backbone - skeleton of application
  • underscore - provides collection's helpers and templates
  • jquery - duh
  • requirejs - js module loader
  • less - CSS pre-processor
  • GRUNT - javascript task runner
Additional libs
  • nvd3 charts - svg charts based on d3.js
  • bootstrap - great UI boilerplate for modern web apps
  • momentjs - nice datetime lib
  • parallaxjs - parallax
  • typeaheadjs - autocomplete
  • jquery-cooki - cookie manager
Unit testing
  • qunit - unit test framework from jQuery team
  • sinon - great UI boilerplate for modern web apps

Used APIs:

  • gd.geobytes.com - location autocomplete
  • api.openweathermap.org - weather data
